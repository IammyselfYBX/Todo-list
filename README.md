# TODO List软件
## 说明
```bash
├── Linux版本    # 是运行在Linux系统的版本软件
│   ├── app      # 可执行程序
│   ├── app.c    # 源代码
│   └── todos.bin # 内容
└── Windows版本  # 是运行在Windows系统的版本软件
    ├── app.c    # 源代码
    └── todolist.exe # 可执行程序
```
